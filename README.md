# MiniManager

A minimal real-time team management dashboard and system certification tracker.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   ```bash
   cp .env.example .env
   ```
   *Note: Add your Firebase credentials to the `.env` file.*

3. Start the development server:
   ```bash
   npm run dev
   ```

## Tech Stack

- React
- Vite
- Tailwind CSS
- Firebase
- dnd-kit
